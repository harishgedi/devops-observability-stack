## 🚀 **Getting Started**

### ⚡ **Quick Start**
```bash
# Clone Repository
git clone https://github.com/harishgedi/devops-observability-stack.git
cd devops-observability-stack

# Deploy with Docker Compose
docker-compose up -d

# Access Dashboards
# Grafana: http://localhost:3000 (admin/admin)
# Prometheus: http://localhost:9090
# Jaeger: http://localhost:16686
# Kibana: http://localhost:5601
```

### 🧪 **Run Tests**
```bash
# Full Test Suite
pytest tests/ --cov=src --cov-report=html

# Production-Grade Tests
python tests/test_production_grade.py

# Security Tests
python tests/security/vulnerability_scan.py
```

### 🚀 **Deploy to Production**
```bash
# Deploy to Staging
python deploy_to_production.py staging

# Deploy to Production (after approval)
python deploy_to_production.py production
```

